Rounded status badge. Semantic tones include the OVP2 knowledge states `durable` / `caveated` / `blocked`. Soft-tinted by default; `solid` fills.

```jsx
<Pill tone="durable">durable</Pill>
<Pill tone="caveated">caveated</Pill>
<Pill tone="blocked">blocked</Pill>
<Pill tone="accent" solid>v0.23.0</Pill>
```
