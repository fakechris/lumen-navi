Outline icon from the Lumen set — 24px grid, 1.8px stroke, round caps, `currentColor`. Lifted verbatim from lumen-cut so it matches the shipping apps exactly.

```jsx
<Icon name="microphone" />
<Icon name="transcript" size={24} />
<Icon name="star" style={{ color: "var(--accent)" }} />
```

Named glyphs: folder, transcript, settings, upload, link, microphone, play, chevronRight, chevronDown, search, star, sun, moon, check, alert, translate, layers, close. Inherits color from `color`. For glyphs outside this set, use Lucide (same stroke weight / rounded outline) and flag it.
